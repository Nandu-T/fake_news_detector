Fake News Detection using Machine Learning and NLP

Overview
This project is a Fake News Detection System developed using Machine Learning and Natural Language Processing (NLP). The model analyzes news headlines and predicts whether the news is Real or Fake.

Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorization
- Logistic Regression

Features
- Text preprocessing
- Feature extraction using TF-IDF
- Fake/Real news classification
- User input prediction

Project Workflow
1. Load dataset
2. Convert text into numerical features using TF-IDF
3. Train Logistic Regression model
4. Predict whether news is real or fake
5. Display prediction results

Installation
pip install -r requirements.txt

Run Project
python fakenews.py

Applications
- Misinformation Detection
- News Verification
- Social Media Monitoring
- Educational NLP Projects
